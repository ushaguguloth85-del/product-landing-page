# Product Landing Page

## Description
This is a responsive Product Landing Page built using HTML and CSS.  
The page is designed with a modern layout and is mobile-friendly.

## Features
- Responsive design for mobile, tablet, and desktop
- Clean and modern UI
- Navigation header and hero section
- Product details section
- Call-to-action button
- Footer section

## Technologies Used
- HTML5
- CSS3
- Flexbox
- Media Queries

## How to Run
1. Download or clone the repository
2. Open the project folder
3. Double-click on index.html
4. The project will open in your web browser
